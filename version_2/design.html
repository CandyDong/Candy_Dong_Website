Xtage is a forum for employees to communicate with each other. Through this platform, employees can submit prospective project requests for internship, ask company-related questions, share recent work accomplishments, post solutions they consider helpful, etc.
Xtage, starting with osqa, an open source framework for Q&A sites, is built with Djangoand mySQL databse.
I redesigned Xtage's look, and turned it from a project prototype into an official website which can be deployed to production. During the process some desired functionalities are successfully implemented, for example, a draggable navigation bar which can be simply customized depending on user preferences, comment/reply/search functionalies, vote for answers/questions, etc, as required by the company.
