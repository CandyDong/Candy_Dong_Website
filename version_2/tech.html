Voco is an assistive app that can translate sign languages into words using data from Myo, a bluetooth-connected wearable armband. We built a classifier using an SVM (support vector machine) model implemented with LIBSVM.
This prototype was able to distinguish 6 words: HELLO, MY, NAME, TARTAN, HACKS, WHATSUP.
Speech recognition was implemented using Google's Speech Recognition API to help people who don't sign communicate better with hearing and speech impaired people.
This is a hackathon project developed during TartanHacks 2018 at Carnegie Mellon University. Team members are Justin Chu, Candy Dong, Bruce Liu, and Jacky Zou.

A Doll's Glove is an glove made for home entertainment. With the glove, users can control a 3d Puppet by moving his/her fingers, and the puppets' limbs will move responsively like an actual puppet being controls through strings.
Driven by Arduino, the flex sensors collect data of each finger joint movement. We used Pyserial python package as a bridge for communication between blender and arduino.
This is a Hackathon project developed during Build18 2018 at Carnegie Mellon University. Team members are Candy Dong, Shirley Zhang, Fan Yan and Anna Yuan.
The blender model after I finished rigging.
In order to present with a holographic effect, we used four different blender view window, each displaying a different angle of the model.
